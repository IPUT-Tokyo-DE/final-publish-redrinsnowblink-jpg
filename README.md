# コズミックきゃっと

## ゲームの内容
プレイヤーは猫を操作して部屋の中を移動し、敵に見つからないようにしながらスコアアイテムを集めるゲームです。
スコアアイテムを集める際にはチャージすることが必要です。
制限時間内に目標スコアに到達するとクリアとなり、時間切れまたは敵に捕まるとゲームオーバーになります。
敵には視界があり、チャージしている最中に視界に入ると敵に掴まりゲームオーバーです。  
プレイヤーは移動しながら敵を避け、効率よくスコアを集めることが目的です。

- W/A/S/Dキー：移動
- 右クリック長押しでチャージ
